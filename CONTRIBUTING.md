# Contributing to Voker

Thank you for your interest in Voker! This project is a collaborative effort between human creativity and AI precision.

## Authorship & Credits
- **Lead Architect & Developer**: Taylor Moon (@taylorcmoon)
- **Primary AI Collaborator**: Gemini CLI

## Contribution Guidelines
1. **Respect Attribution**: All contributions must respect the established authorship of Taylor Moon.
2. **AI-Assisted Workflows**: We encourage the use of AI tools for implementation, provided they are guided by clear human intent and thorough validation.
3. **Commit Standards**: Use descriptive commit messages. If a change was implemented by an AI agent, please note the agent's name and the user's directive.
